# My Markdown Example

## Introduction

This is a brief introduction to **Markdown**. It is a lightweight markup language with plain-text formatting syntax.

### Features of Markdown

- Easy to read and write
- Supports various elements like headings, lists, links, and images

## Headings

You can create headings by using the `#` symbol:

# H1
## H2
### H3
#### H4

## Lists

### Unordered List
- Item 1
- Item 2
  - Subitem 2.1

### Ordered List
1. First item
2. Second item
3. Third item
